## 📥 Installation

### Depuis le Chrome Web Store

L'extension sera prochainement disponible sur le Chrome Web Store.  
*(Lien à ajouter dès la publication)*

### Installation manuelle

1. Téléchargez le dernier fichier ZIP depuis la section [Releases](https://github.com/Damien-Codes/QuantScale/releases).
2. Extrayez le contenu du ZIP dans un dossier de votre choix.
3. Ouvrez Chrome et accédez à `chrome://extensions/`.
4. Activez le **Mode développeur** (coin supérieur droit).
5. Cliquez sur **Charger l'extension non empaquetée** et sélectionnez le dossier extrait.

L'extension est maintenant installée ! Une icône apparaîtra dans la barre d'outils.
