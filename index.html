Télécharger ZIP

Instructions d'installation manuelle

  1. Téléchargez le fichier ZIP depuis GitHub
  2. Extrayez le contenu du ZIP dans un dossier
  3. Ouvrez Chrome et allez dans chrome://extensions/
  4. Activez le "Mode développeur" en haut à droite
  5. Cliquez sur "Charger l'extension non empaquetée" et sélectionnez le dossier extrait
